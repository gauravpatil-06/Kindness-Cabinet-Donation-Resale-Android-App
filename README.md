## 🌟 Introduction

**Kindness Cabinet** is built with a simple idea:  
> *“Connecting people through kindness and reducing waste.”*

This application helps users donate or resell unused items so that they can be reused instead of being wasted. It creates a digital platform where people can easily give, receive, and support each other.

This is **not just a college project** — it is a social-impact application focused on sustainability, community support, and smart resource sharing.

---

## 🚀 How It Works

*   **Easy Login**: Users can sign in using **Google Sign-In** or **OTP verification** for quick and secure access.
*   **Add Items**: Users can upload items with images, select category, and choose donation or resale.
*   **Browse & Search**: Users can explore items using filters like category, price, and location.
*   **Chat System**: Built-in chat allows direct communication between donor and receiver.
*   **Track Activity**: Users can track their donations, purchases, and overall contribution.

🌱 This platform allows users to:
- Donate or resell items digitally

## ⚙️ Installation & Setup

### 1. Repository Setup
```bash
git clone https://github.com/gauravpatil-06/Kindness-Cabinet-Donation-Resale-Android-App.git
cd Kindness-Cabinet-Donation-Resale-Android-App
```

### 2. Environment Configuration
- Open the project in **Android Studio**
- Sync Gradle and install required dependencies
- Add Firebase configuration:
  - Place `google-services.json` inside the `app/` directory
- Update the backend API URL in the constants file
